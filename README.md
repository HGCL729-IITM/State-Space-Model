# State-Space-Model
The generated zonal maps of precipitation and temperature for the Indian subcontinent are available at a spatial resolution of 0.25°. These datasets are provided for both cases, with and without integration of topographical features and are accessible in (.gpkg) and (.xlsx) file formats. Codes used in this study are available upon request: sreeparvathyvijay@civil.iitm.ac.in
